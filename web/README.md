# CodeFlip

## Setup

To set up this project, a realt-time database, authentication and cloud storage must first be set up with Firebase. This guide shows how to do it: https://www.robinwieruch.de/complete-firebase-authentication-react-tutorial/

Then, the project is ready for setup. 
From the web directory, install all dependencies using the packet manager npm:

### `npm install`

To run the project:

### `npm start`

The game will now run in development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
